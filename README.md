# 3D Maze Game – Unity Personal Project

A small 3D maze prototype built in Unity with 3 playable levels, featuring checkpoints, level progression, game-over logic, timer/score systems, and mobile-friendly controls.

## Features
* 3 progressively difficult maze levels
* Mobile joystick movement controls
* Cinemachine follow camera
* Timer and score system
* Checkpoint & respawn system
* Death zones with Game Over UI
* Level restart and progression system
* Main Menu with level selection
* Basic VFX and Trail Renderer feedback
* Level Complete UI with final time & score

## Built With
* Unity
* C#
* TextMeshPro
* Cinemachine

## What I Learned
This project helped me practice:
* Scene management
* UI systems
* Singleton-based game flow
* Trigger/collision handling
* Respawn & checkpoint systems
* Timer and score logic
* Organizing gameplay systems across multiple scenes
